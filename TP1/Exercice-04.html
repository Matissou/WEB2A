Fiche balises HTML

Niveaux de titre

  • <h1> : titre de niveau 1
  • <h2> : titre de niveau 2
  • <h3> : titre de niveau 3
  • <h4> : titre de niveau 4
  • <h5> : titre de niveau 5
  • <h6> : titre de niveau 6

Balises de base

  • Balises
    • <p> : paragraphe
    • <br> : retour à la ligne
    • <adress> : adresse
    • <strong> : important
    • <em> : emphase
    • <a> : lien
    • <img> : image
    • <div> : balise générique d'un bloc
    • <header> : en-tête
    • <footer> : pied de page
  • Attributs
    • id="id" : identifiant unique de la balise
    • class="class" : classe de la balise (pas forcément unique)

Listes

  • <ul> : liste non ordonnée
    • <li> : élément de liste non ordonnée
  • <ol> : liste ordonnée
    • <li> : élément de liste ordonnée
  • <dl> : liste de définitions
    • <dt> : terme a définir
    • <dd> : définition du terme

Tableaux

  • Balises
    • <table> : déclaration d'un tableau
    • <thead> : en-tête du tableau
    • <tbody> : corps du tableau
    • <tr> : ligne du tableau
    • <th> : cellule d'en-tête
    • <td> : cellule
  • Attributs
    • rowspan="NB" : nombre de cellules a fusioner en colone
    • colspan="NB" : nombre de cellules a fusioner en ligne

Formulaires

  • Balises
    • <form> : déclaration d'un formulaire
    • <input> : champ texte sur une ligne
    • <textarea> : champ texte sur plusieurs lignes
    • <select> : liste déroulante
    • <option> : élément d'une liste déroulante
  • Attributs
    • name="name" : nom du formulaire
    • value="value" : valeur donnée au serveur lors de la soumission du formulaire
    • size="NB" : largeur d'un champ texte
    • type="type" : type peut etre égal à text, radio, checkbox ou submit pour le bouton d'envoi
